👨‍🏫 Registro de Docentes y Personal

Formulario para que los docentes llenen sus propios datos

Una sola vez al instalar el sistema (o cuando ingresa un nuevo docente). Al aprobar la solicitud, se crea automáticamente la cuenta de acceso y se envían las credenciales por WhatsApp.

Enviar por WhatsApp 👁 Vista previa 📋 Ver solicitudes
Datos que se solicitan: Nombre completo · Cédula · Email · Teléfono/WhatsApp · Especialidad · Título académico · Años de experiencia · Carta de presentación

🎓 Inscripción de Nuevos Estudiantes

Formulario público para padres de familia — aspirantes nuevos

Se comparte cada inicio de año lectivo con las familias interesadas. El representante llena los datos del estudiante. La secretaría revisa y aprueba. Al aprobar, se crea la cuenta del estudiante automáticamente.

Enviar por WhatsApp 👁 Vista previa 📋 Panel secretaría ⚙️ Ver en Admin
Datos que se solicitan: Nombre del estudiante · Cédula · Fecha de nacimiento · Año al que aspira · Instrumento elegido · Datos del representante (nombre, email, teléfono) · Documentos requeridos (cédula, certificados, foto)

🔄 Renovación de Matrícula (flujo anual)

Formulario para estudiantes ya registrados — renovar cada año

El estudiante (o su representante) ingresa con su cuenta existente y solicita la renovación para el nuevo ciclo lectivo. El sistema detecta automáticamente el año siguiente. Este link se comparte cada año al inicio del período de matrículas.

Enviar mensaje de renovación por WhatsApp 🔍 Seguimiento de solicitudes
Requiere cuenta previa. El estudiante inicia sesión y sube los documentos de renovación. El sistema avanza automáticamente al siguiente año (ej. de 3° a 4°).

📋 Resumen del flujo

Quién Cuándo Flujo
👨‍🏫 Docente / Personal Una sola vez (al instalar o al contratar) Recibe link → llena formulario → Admin aprueba → cuenta creada + credenciales por WA
🎓 Estudiante nuevo Inicio de cada año lectivo Representante recibe link → llena datos + sube documentos → Secretaría aprueba → cuenta creada + credenciales por WA
🔄 Estudiante existente Inicio de cada año lectivo Inicia sesión con cuenta → link de renovación → sube documentos → Secretaría aprueba → matrículas actualizadas
🗑️ Docente que se va Al dejar la institución Admin va a Admin → Docentes o Admin → Usuarios → eliminar o desactivar la cuenta
